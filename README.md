# pyStock
stock app in python


# Prerequisites

- Python3 

Clone Repo

```
git clone git@github.com:spuzzelsnest/pyStock.git

cd pyStock

```

## Creating a python 3.9 environment

```

python3.9 -m venv venv

venv/bin/activate


```

## install modules
- pandas
- matplotlib
- seaborn
- requests
- yfinance
